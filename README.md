# Olas Operate
Electron + NextJS + Flask application to one-click run Agents.

## Technologies Used
- NextJS
- Electron
- AntD
- TypeScript
- Python
- Flask
- Poetry


## Getting Started

### Install Yarn 

```bash
npm install --global yarn
```

### Install dependencies

```bash
yarn
```

### Run the development app

```bash
yarn dev
```
